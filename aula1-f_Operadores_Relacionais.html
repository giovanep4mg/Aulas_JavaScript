Operadores Relacionais no JS

Usados para fazer verificações:
Se é maior ou menor,se é igual ou diferente.
Se é maior ou igual, se é menor ou igual.

Vamos fazer essa verificação, e vai dá a resposta:
true é verdadeiro.
false é falso .

Para ajudar a visualização de como fica, adicionei um botão que vai dá a resposta.

Exemplo de código:
Abra a tag "script" e dentro dele coloque:

// 5 é maior que 4
window.alert(5 > 4);
Agora veja como seria, clicando no botão abaixo:


// 5 é menor que 4
window.alert(5 * 4); * por que não tem como colocar o sinal de menor aqui
Agora veja como seria, clicando no botão abaixo:


// 5 é igual a 4 => sinal de igual >> usa dois simbolos de iguais "=="
window.alert(5 == 4);


// 5 é maior ou igual a 5
window.alert( 5 >= 5 );
Agora veja como seria, clicando no botão abaixo:


// 8 é menor ou igual a 12
window.alert( 8 *= 12 ); * por que não tem como colocar o sinal de menor aqui
Agora veja como seria, clicando no botão abaixo:


// 8 não é igual a 9 => diferente, usa esses simbolos "!=" dizendo que não é igual
window.alert(8 != 9);
