Extrair Elementos

Extrair elementos podem ser:
caracteres, simbolos, números, espaços, etc...
Para corta pedaços de texto.

slice = vai corta os caracteres passado pelo usuário "( )".
dentro do "()", você passa quantos caracteres você quer corta do texto.

No primeiro exemplo, usamos o texto => "Aprendendo JavaScript",
vamos passar o parâmetro (10), vai corta os 10 primeiros caracteres.
Então terá que aparecer na tela = JavaScript .

No segundo exemplo, usamos o mesmo texto => "Aprendendo JavaScript",
vamos passar o parâmetro (10, 15 ),
vai corta os 10 primeiros caracteres => "Aprendendo".
depois de 15 caracteres será cortado denovo => "Script"

Então será exibido, somente o que estiver depois do "10" antes do "15",
a parte que eu quero é depois de 10 caracteres e antes de 15 caracteres.

Aviso: o espaço também conta como caractere !!


Exemplo de código:

Abra a tag "script", dentro dela coloque:

//variavel
var frase = "Aprendendo JavaScript";


// vai extrair da frase os primeiros "10 caracteres",
// que será o => "JavaScript"
alert(frase.slice(10));

// vai extrair os primeiros "10" caracteres, depois
// extrair os que estão depois de "15" caracteres.
alert(frase.slice(10, 15));

Fecha a tag "/script"
