Atributos

Atributos são complementações, aparece depois da tag.

Usaremos como exemplo a tag a seu atributo será o href,
que seria um link para algum site, página,

Iremos fazer a troca desse atributo "a página do site",
quando você clicar em um botão, o link será trocado.

Quando clicar, ai invés de ir para um lado o link te levará para outro lado.


Primeiro vamos criar um link, dentro da tag "body",
pode ser qualquer link só para teste, usando a tag "a":

a href="https://www.facebook.com/giovani.martins3/">clique aqui

Depois crie um "button", com evento "onclick" que vai acionar a função 'alterarLink()':

button onclick="alterarLink()"
Veja como é o código:

Agora vamos criar a função:
Exemplo de código:

Abra a tag "script", dentro dela coloque:

__//funçao.
__function alterarLink() {


___// obter um link >>
___// salva na var "link", o que tem na tag "a", na posição "0"
___var link = document.getElementsByTagName("a")[0];


_______// pega esse novo link e salva na var link
_______link.href = "https://github.com/giovanep4mg";


__ }
