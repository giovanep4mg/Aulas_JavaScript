Seletor através do HTML

Para selecionar o HTML de alguma tag, precisa:

Criar uma variável para guarda essa informação:
var texto = *que vai receber :

Diz que o que vai receber é um documento:
document. *ai você especifica o que você quer:

Diz para pegar o elemento, que é uma "tag" e vai passar o nome dessa tag:
getElementsByTagName() *ai dentro do parentes coloque o nome da tag:

Diz qual é o nome da tag e qual é sua posição:
("h1") [0] a tag é o "h1" e sua posição é o "0" vai pegar o primeiro 'h1'

Ficando assim o comando para pegar algo de um HTML :
var texto = document.getElementsByTagName("h1")[0];

Vamos começar criando 3 tipos de "h1":

h1 Proway /h1
h1 Seletor de JS /h1
