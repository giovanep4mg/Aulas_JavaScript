Estrutura base do JavaScript

Essa é a estrutura base do JS.
Dentro da tag head você adiciona a tag do JavaScript => script.

Dentro da tag script será adicionado os códigos java Script.

Caixa de mensagem


Caixa de mensagem, é essa janelinha, para dá um aviso ao usuário, orienta ele no site.
Por exemplo, o usuário não preencheu todo o formulário, ai aparece essa caixa de mensagem.
Avisando sobre algo, que você fez ou deve fazer ou deixou de fazer.
