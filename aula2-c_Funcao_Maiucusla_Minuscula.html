Maiúscula e minúscula

Deixa as letras do texto Maiúscula ou Minúscula.

toUpperCase = deixa todas as letras maiúsculas.

toLowerCase = deixa todas as letras minúsculas.

Usado como um filtro, para deixar tudo de um jeito só,
exemplo o usuário digita o email ou senha com uma letra maiúscula,
daria erro, por isso se usa esse filtro para ajudar a não dá esse erro.


Exemplo de código:

Abre a tag "script", dentro dela coloque :

//variável
var palavra = "INformáTiCa";


//exibir com letra maiúscula
alert(palavra.toUpperCase());


//exibir com letra minúscula
alert(palavra.toLowerCase());
