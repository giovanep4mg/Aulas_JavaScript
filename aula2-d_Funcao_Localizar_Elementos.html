Localizar Elementos

Vai te dá a posição da letra, conta apartir de 0,1,2,3....

Então se retorna 0 é 1, se for 2 é 1, se for 3 é 2...
Usada para fazer validações, emails,....

indexOf conta aparti do inicio.
lastIndexOf conta de trás para frente.


Exemplo de código:

Abra a tag "script", dentro dela coloque:

//variavel
var nome = "Giovani";


//retorna a primeira letra "i"
alert( "A posição do 'v' é : "+nome.indexOf("v"));;

//retorna a última letra "a"
alert( "A posição do 'a' é : "+nome.lastIndexOf("a"));
