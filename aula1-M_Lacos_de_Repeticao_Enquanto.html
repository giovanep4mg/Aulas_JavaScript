Laço de Repetição Enquanto "WHILE"

Para executar uma ação várias vezes.
Atenção: tem que fazer a implementação, para não dá loop infinito.
Para fazer validações, para buscar informações em um banco de dados.

No exemplo:
Executar o "oi" 3 vezes.


Exemplo de código:
Abra a tag "script" dentro dela coloque:

//variável
var indice = 1 ;


// Laço de repetição
while (indice <= 3) {

// enquanto o indice for maior ou igual a 3, exibirá essa mensagem.
_________alert("oi"+indice);

indice++; * implementa o indice, vai aumentar o valor dele para parar o loop *
}
