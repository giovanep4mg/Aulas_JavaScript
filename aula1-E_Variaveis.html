Variáveis no JS

São palavras para armazerna, tipos de dados como: palavras, números, lógico

Atenção:
Não utilize caracteres especiais.
Não começa com números, pode colocar depois.
Não coloque as palavras separadas, deve ser assim: "tipoAssim","casaNumero"...
Para números decimais, utilize o ponto ".", e não a vírgula ",".


Exemplo de código:

Depois da tag "script" coloque esses códigos:

"// Variáveis" duas barras é para marca que é um comentário.
var nome = "Giovani"; a variável nome recebe o nome 'Giovani'.
var idade = 28; a variável idade recebe o número '28'.
var altura = 1.67; a variável altura recebe o número '1.67'.
var estuda = true; a variável estuda recebe o número 'true' ou seja "verdadeiro".


Para ser exibido na página, na "janelinha" será mostrado uma variável após a outra:

//para exibir na tela.
window.alert("Seu nome: "+nome);
window.alert("Sua idade: "+idade);
window.alert("Sua altura é: "+altura);
window.alert("Você estuda: "+estuda);
