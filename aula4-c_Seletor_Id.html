Seletor através de Classe

Vai copiar o dado digitado no campo de texto na página e exibir na "janelinha".

Depois apaga o campo de texto, da página deixando em branco.

Não precisa colocar qual é a tag, somente o "id"

Quando for uma tag de formulário, para exibir usa o => *variável.value



Vamos começar criando um "input" para o usuário digitar,
dentro desse input terá o "type = text" e seu "id = campo":

input type="text" id="entrada"

Depois vamos criar um "button", e ele terá um evento "onclick",
que executar uma "Função" que vai exibir o conteúdo digitado:

button onclick="exibirConteudo()" "Clique aqui" /button

Agora vamos criar a "função" => "exibirConteudo()", na tag script.

Exemplo de código:

Abra a tag "script", dentro dela coloque:

___//funçao.
___function exibirConteudo () {


________//obter dados do "id campo", no id NÃO precisa passar a posição "[?]"
________var campo = document.getElementById("campo");


______//exibir o conteúdo => como é tag é "input" usa => "value" e não o innerHTML
______alert(campo.value);


_______// vai deixar vazio o "campo", está enviando para o "campo"
_______campo.value = "" ;

__ } fecha a "function"
