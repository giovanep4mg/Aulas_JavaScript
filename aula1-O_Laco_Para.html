Laço de Repetição Para "for"

Você tem que colocar esses 3 parâmetros:
var indice = 1 => INICIO => coloca uma variável e seu valor .
indice <= 5 => vai repetir até o indice ser maior ou igual a 5.
indice++ => acrescenta +1 ao valor da variável indice, fazendo ele parar de rodar.

Para vermos melhor vamos adicionar uma caixa de mensagem, avisando quanto está valendo o "indice".


Exemplo de código:
Abra a tag "script", e coloque dentro dela:

//laço de repetição
for(var indice = 1; indice <= 5; indice++){
alert("O indice vale:"+indice);
}
