Vetores

Vetor = são várias informações dentro de uma variável, um banco de dados temporário.

Para mostra um item dentro do vetor, usa o "[]", passando o número de qual você quer.
A ordem dentro do vetor começa com zero "0", ou seja,"HTML" é "0", "CSS" é "1" ...


Exemplo de código:

Abra a tag "script", dentro dela coloque:

//variável recebe => essas pequena lista => "HTML", "CSS", "JAVASCRIPT"
var cursos = ["HTML", "CSS", "JAVASCRIPT", ];


// caixa de mensagem, exibirá o curso "0", que é o HTML
alert(cursos[0])

//exibirá o curso "2", que é o JAVASCRIPT
alert(cursos[2]);
