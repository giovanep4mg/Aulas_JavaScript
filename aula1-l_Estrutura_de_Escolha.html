Estrutura de escolha "SWITCH"

Para executar ações, quando você tem várias opções.
Como um menu, várias coisas, que cada uma dá uma resposta diferente.

Não usa os condicionais relacionais (maior q,menor q,igual q...).
Pode ser usado com condicionais que usam "if".

Como exemplo vamos usar um menu de restaurante, que a comida será pedida pelo seu código,
vai ter algumas opções e pelo seu código você escolhe qual você quer.


Exemplo de código:
Abra a tag "script" dentro dela coloque:

var codigo = 1; => o código será "1"


//escolha
switch(codigo) { vai receber o código que você passou
