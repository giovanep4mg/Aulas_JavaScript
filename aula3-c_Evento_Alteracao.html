Evento Alteraração

onchange => ao alterar, executa assim que acontece uma ação.
Cria uma caixa de seleção, com algumas opções.
Vai capturar o que foi selecionado, e mostra ao usuário na "janelinha".

Vamos criar um campo de formulário usando a tag select,
dentro da tag select adicione :
select onchange="mensagem()" id="cursos"
" onchange="mensagem()" " => vai acionar a função.
" id="cursos" " => para saber o que o usuário escolheu.

dentro dela colocamos algumas opções usando as tags "option":
option "Escolha seu curso" /option
option "HTML" /option
option "CSS" /option
option "JAVASCRIPT" /option
