Arredondar Valores

Quando digitamos o Math, que é usado para trabalhar com matemática,
teremos várias funções, nos veremos o Math.round que é usado para arredondar números.

Vamos criar duas variáveis:
"var nota = 7.50;"
"var nota1 = 7.49;"

Depois um alert,
Dentro do alert coloque o Match.round(nota),
como parâmetro está passando a nota, o valor que está dentro da nota.


Acima de 0.5 => arrendonda pra cima.
Abaixo de 0.4 => arrendonda pra baixo.


Exemplo de código:

Abra a tag "script", dentro dela coloque:

//variavel
var nota = 7.50;
var nota1 = 7.49;


//exibir a nota, arredondando para cima
alert(Math.round(nota));


//exibir a nota, arredondando para baixo
alert(Math.round(nota1));
