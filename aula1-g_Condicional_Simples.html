Condicionais Simples no JS

Usados para fazer verificações de acordo com as condições.
Passa uma regra, uma comparação e ela diz a respota de acordo com a condição.

Exemplo:
A pessoa tem 17 anos, no programa irá fazer uma verificação,
se ela pode ou não tirar a carteira, e vamos colocar uma condição que é,
se for maior ou igual a 18 anos vai receber um Aprovado, senão vai receber um Reprovado.


Exemplo de código:
Abra a tag "script", dentro dela coloque:

//variável
var idade = 17 ;

//se (idade for maior ou igual a 18)
if(idade >= 18){

//exibi essa mensagem
alert("Aprovado.")

//senão
} else{
//exibi essa mensagem
alert("Reprovado.")
}
