Operador Lógico OU "||"

Usados para comparar duas ou mais informações.
Trabalha junto com condicional, principalmente "if".
É bem mais maleável dando certo só uma condição ele já vai mostrar

Como exemplo, vamos pegar a nota de dois trabalhos de um aluno,
que para passar precisar tirar a nota "7", cada trabalho já vale "7",
então se algum dos dois trabalhos ele tirou "7", ele vai ser Aprovado.

Vamos para o código:
Exemplo de código:
Abra a tag "script", e dentro dela coloque:

//variável var trabalho1 = 7; var trabalho2 = 5;

//se (trabalho1 for maior ou igual a 7) OU (trabalho2 for maior ou igual a 7)
if((trabalho1 >= 7) || (trabalho2 >= 7)){
_______alert("Aprovado!");
}else{
_______alert("Reprovado!");
}
