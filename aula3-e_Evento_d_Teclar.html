Evento de Teclar

Existem apenas 3 eventos de teclados no Java Script, são eles:

onkeyUp
Exibe o caractere e depois executa a ação.

onKeyDown
Executa a ação e depois exibi o caractere.

onKeyPress
Executa a ação de depois exibi o caractere.
Não tem suporte as seguintes teclas:
F1..F2,ALT,CTRL,SHIFT,....

Exemplo de código:

No html, no "body", vamos adicionar um campo de texto para digitar, para isso
adicione um "input", do tipo "text", com id "campo", e colocaremos o evento nele.

input type="texto" id="campo" onkeyup="exibir()"

O evento é o "onkeyup", que vai acionar a função "exibir()"
Veja como é o código:


SCRIPT abra a tag "script", dentro dela coloque :

__//função.
__function exibir (){

_________// Obter o valor que foi digitado no campo de texto
_________var campo = document.getElementById("campo").value;


__________//exibir o conteúdo digitado no campo
__________alert(campo);

__}

Veja como é o código:


onkeyUp
Essa função => Vai pegar a primeiro caractere digitado, mostrar na "janelinha",
fará isso toda vez que for digitado um caractere, e somando ao que já foi digitado.





onkeydown
Essa função => Vai pegar a primeiro caractere digitado, NÃO vai mostrar na "janelinha",
SÓ vai mostrar depois que for digitado outro caractere.
fará isso toda vez que for digitado um caractere, e somando ao que já foi digitado.
